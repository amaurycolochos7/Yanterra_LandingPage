# Yanterra - Llantera Monumento 24 Hrs

Sitio web estático para Llantera Monumento en García, N.L.

## Instrucciones de Despliegue

### 1. Subir a GitHub
Para desplegar en Vercel, primero debes subir este código a un repositorio de GitHub:

1. Crea un nuevo repositorio en GitHub (ej. `yanterra-web`).
2. Sigue los comandos para "push an existing repository":
   ```bash
   git remote add origin https://github.com/TU_USUARIO/yanterra-web.git
   git branch -M main
   git push -u origin main
   ```

### 2. Desplegar en Vercel
1. Ve a [vercel.com](https://vercel.com).
2. Haz clic en **Add New...** > **Project**.
3. Importa el repositorio `yanterra-web`.
4. Haz clic en **Deploy**.

¡Eso es todo! Tu sitio estará en línea.
